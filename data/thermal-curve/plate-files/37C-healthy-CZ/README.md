Raw data from the resazurin assay performed under repeated heat stress at 37 °C on Pacific oyster seedlings. 
Plate files were run by Christina Zhang. 
Files named platex_Tx correspond to 37 °C–treated, healthy oysters from round 1, run on 2/7/2026.
Files named platex_Tx_R2 correspond to 37 °C–treated, healthy oysters from round 2, run on 2/7/2026.
